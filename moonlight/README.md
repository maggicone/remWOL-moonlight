# What is remWOL-Moonlight?

remWOL-Moonlight is a slightly modified version of [Moonlight-QT](link-to-moonlight-qt) designed to work with the [remWOL project](link-to-remwol-folder).

It features a new button **Wake PC (API)** that can send an API request directly from Moonlight.  
The API request is configurable via the Moonlight settings.

The goal of this project is to eliminate the need for dedicated extra software to wake up your PC when you are not at home.  
Just open Moonlight and wake your PC from anywhere in the world.

# Installation

## - Windows

> [!IMPORTANT]  
> You may need to uninstall any existing version of Moonlight first.

Download and install the Windows executable [here](link-to-releases).

## - SteamOS / Linux

Switch to **Desktop Mode**, download the AppImage [here](link-to-releases), and place it wherever you prefer.

Then:

Right click the file → **Add to Steam**

You can now switch back to **Gaming Mode**.

---

# Usage

Open **remWOL-Moonlight**, go to **Settings**, and scroll down to the new **Wake on LAN** section.

Enable the checkbox and fill in the fields with your configuration values.

If everything has been set up correctly, you will now be able to wake your PC from anywhere by clicking the new **Wake PC (API)** button that appears in the dropdown menu when you select an offline computer.

> [!IMPORTANT]  
> Currently, only one computer can be configured for the **Wake PC (API)** request.  
> If you have multiple devices paired with Moonlight, pressing **Wake PC (API)** will always send the request to the same configured computer.
